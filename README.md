# Olist-E-commerce-Global-Analytics-Suite-project
# 🛒 Olist E-Commerce Global Analytics Suite

A comprehensive end-to-end data analytics project that explores the Olist Brazilian E-commerce dataset to uncover business insights, customer purchasing behavior, sales performance, and operational trends. This project demonstrates the complete analytics lifecycle, from raw data preparation and cleaning in SAS to business reporting and case study documentation.

---

## 📌 Project Overview

The Olist E-Commerce Global Analytics Suite was developed to transform raw transactional data into meaningful business intelligence. The project applies data analytics techniques to identify sales trends, customer behavior, product performance, and operational efficiencies that support data-driven decision-making.

The project showcases practical skills in:

- Data Cleaning
- Data Transformation
- Data Analysis
- Business Intelligence
- Statistical Reporting
- Data Documentation
